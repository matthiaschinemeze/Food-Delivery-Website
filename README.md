Food Delivery & Logistics Website
A modern, responsive Food Delivery and Logistics web application built with HTML, CSS, and JavaScript. The platform is designed to provide users with a seamless food ordering experience—from discovering restaurants and browsing menus to managing their cart and checking out. The project also serves as a foundation for integrating a backend, authentication, online payments, and real-time order tracking.

✨ Features
Responsive and modern user interface
Interactive homepage with featured restaurants
Popular dishes section
Search bar for finding dishes
Sliding cart sidebar
Shopping cart management
Checkout page
User authentication pages (Login & Sign Up)
Restaurant menu navigation
Clean and intuitive user experience
Mobile-friendly design
